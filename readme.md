## What I work on

**LLM Engineering & Agents**
Fine-tuning open-source models (Mistral 7B with LoRA/QLoRA), building ReAct agents with LangGraph, designing RAG systems with ChromaDB and Amazon Bedrock, and orchestrating multi-step AI pipelines with LangChain.

**MLOps & LLMOps**
End-to-end model lifecycle management with MLflow, model serving via FastAPI, containerization with Docker, infrastructure as code with Terraform, and LLM observability with Langfuse and RAGAS evaluation.

**Deep Learning & NLP**
Multimodal classification with TensorFlow/Keras and transfer learning (MobileNetV3), sentiment analysis benchmarking classical NLP vs. DistilBERT/HuggingFace Transformers, and computer vision pipelines.

**Machine Learning**
Regression and classification with scikit-learn, ensemble methods (Gradient Boosting, Random Forest), hyperparameter tuning with GridSearchCV, and SHAP-based model explainability.
